The Background property is used for defining the background
effects of the particular element in the HTML. This is a shorthand
property of CSS because we can add, resize, repeat a particular image
using the Background property at once in CSS. We can use the Background
property in CSS in three ways using inline CSS, internal CSS, and external
CSS.
